# Support

Use the repository issue templates for bugs and feature requests.

- Bug reports: use the bug report template.
- Feature requests: use the feature request template.
- Security issues: do not use public issues; follow [`SECURITY.md`](/Users/vanna/Source/devplat/SECURITY.md).

Operational and implementation-specific follow-up should stay in pull request review threads when it is directly tied to a change under review.
