# DevPlat

DevPlat is a Discord-first autonomous development platform built as a strict native-ESM TypeScript monorepo. It combines auditable platform packages, a thin OpenClaw adapter, generated JSON schemas, and deployment surfaces for Docker, Helm, and GitHub Pages.

## What You Get

- Contract-first packages with sibling tests and generated schemas
- OpenClaw adapter coverage for the full DevPlat capability surface
- Discord-oriented task, approval, and thread orchestration flows
- Docker and Helm packaging for self-hosted runtime deployment
- SonarCloud-aware CI with strict repository validation
